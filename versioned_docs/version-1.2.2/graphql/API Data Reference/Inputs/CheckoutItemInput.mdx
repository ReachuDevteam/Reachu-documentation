```javascript
type CheckoutLineInput {
  productId: ID
  variantName: String
  quantity: Int
}
```

#### Fields

**`productId (ID)`**

The id of the added product to the checkout.

**`variantName (String)`**

The id of the added variation to the checkout.

**`quantity (Int)`**

The number of items purchased.

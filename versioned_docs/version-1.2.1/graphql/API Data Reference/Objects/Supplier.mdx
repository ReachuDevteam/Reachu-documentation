<font color="grey" size="4">

Represents data of the user's type: supplier.

</font>

```javascript
type Supplier {
  id: ID
  name: String
  avatar: String
  warehouse: Address
  return: Return
}
```

Fields

**`id (ID)`**

**`name (String)`**

The comercial name of the supplier

**`avatar (String)`**

This is a url with a link to the logo of the supplier, which is uploaded to the
Outshifter server.

**`warehouse (Address)`**

Address of the warehouse of the supplier.

**`return (Return)`**

Some key characteristics of the return terms of the supplier.

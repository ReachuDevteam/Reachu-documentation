_Alternatively_:

- [ ] Click on **Plugins** in your WordPress sidebar menu.

- [ ] Look for the **Add New** button, and press it.

- [ ] In the plugin search tab, write “Reachu Export” and click **Search**.

- [ ] Click the **Install Now** button and wait for it to install correctly.

- [ ] Once it has been installed you must click the **Activate** button.

- [ ] Then, click on **WooCommerce** in your WordPress sidebar menu.

- [ ] Go to **Settings**. Within Settings, click on the tab called **Reachu**.

- [ ] Log in using the same API Key of your Reachu account.

You will find the API Key in the settings section of your Reachu profile.

- [ ] Go to https://dashboard.reachu.io/login and log in using your Reachu
      username and Password.

- [ ] Head over to the **Settings** section by pressing the icon in the top
      right corner.

- [ ] Select **Sales Channels**.

- [ ] Then **copy the API Key** (in the event that there is no API Key
      generated, press the create button, then copy the Key.

- [ ] Choose your default currency and click on **Save Changes** button to save
      the configuration

:::info

Make sure you have created an Reachu Channel account. To apply for an account,
please fill out [this form](https://dashboard.reachu.io/register).

:::

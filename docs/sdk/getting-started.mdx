# Getting Started

## Installation

Install Reachu's sdk via npm

```html
npm i @reachu/sdk
```

## Query Parameters

| Name                     | Required   | Description                                                                                                      |
| :----------------------- | :--------- | :--------------------------------------------------------------------------------------------------------------- |
| `authToken`              | `required` | The `authToken` string you receive when you [Add a Participant](/api/#/operations/add_participant) to a meeting. |
| `orgId`                  | `optional` | Pass your organization if you want to use your custom presets.                                                   |
| `showSetupScreen`        | `optional` | Whether to show the Setup Screen Defaults to `true`.                                                             |
| `disableVideoBackground` | `optional` | Whether to disable the video background changer button on the control bar. Defaults to `false`.                  |
| `endRedirectURL`         | `optional` | The URL to redirect to on end of meeting                                                                         |

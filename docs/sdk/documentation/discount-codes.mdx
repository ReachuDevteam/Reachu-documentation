# Discount Codes

Discount codes are a great way to incentivize a purchase! You can activate a
discount coude for the whole cart.

:::info

:::

```js
async function deleteCart(CART_ID) => {
  const sdk = getSDK();
  await sdk.shopCart.cart.delete(CART_ID);
};
```

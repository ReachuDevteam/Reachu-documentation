## Explore Reachu GraphQL API with Apollo Sandbox

Dive into the capabilities of the Reachu GraphQL API using
[Apollo Sandbox](https://studio.apollographql.com/sandbox), the powerful
in-browser IDE designed for developers to write, validate, and test GraphQL
queries. Apollo Sandbox provides an intuitive interface for exploring the full
potential of Reachu's API without the need for any setup or additional tools.
Whether you're a seasoned developer or just starting with GraphQL, Apollo
Sandbox simplifies the process of understanding and utilizing the Reachu API.

### Getting Started with Apollo Sandbox

1. **Access Apollo Sandbox:** Begin by visiting
   [Apollo Sandbox](https://studio.apollographql.com/sandbox) to access the IDE
   directly in your browser.

2. **Connect to Reachu's GraphQL Endpoint:** Enter the Reachu GraphQL endpoint
   URL in Apollo Sandbox to start exploring the API. Here's the endpoint you'll
   need:

   ```plaintext
   https://graph-ql-dev.reachu.io/
   ```

When a product has a variable status, it has options.

</font>

```javascript
class ProductOption {
  name: String
  value: String
}
```

#### Fields

**`name (String)`**

The name of the option.

**`value (String)`**

The values of the option. As it is a string, each value must be separated by a comma.

#### Example

```javascript
{
  name: "Size"
  value: "Small,Medium,Large"
}
```

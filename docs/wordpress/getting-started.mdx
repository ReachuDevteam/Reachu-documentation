## Introduction

Reachu is the API for product data access. Channels can use Reachu to get access
to product data from multiple Suppliers.

The
“[Reachu Embed Commerce](https://wordpress.org/plugins/outshifter-embed-commerce/)”
plugin lets Channels connect their WordPress with Outshifter to start selling
products on their WordPress pages and posts. It enables Channels to select and
automatically import products from Outshifter to their WordPress with one click.
Outshifter ensures that product data, stock, fulfillment, and shipping
information is synchronized with each of the Suppliers eCommerce systems.
